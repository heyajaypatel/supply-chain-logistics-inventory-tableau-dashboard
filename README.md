# Supply Chain Logistics & Inventory Efficiency Dashboard

## Project Overview

This project focuses on monitoring logistics operations and inventory efficiency using Tableau. The dashboard provides insights into delivery performance, supplier efficiency, inventory levels, stockout risks, and transportation costs to support operational decision-making.

---

## Objective

The objective of this project is to:

* Monitor logistics and delivery performance
* Track inventory efficiency and stock levels
* Identify supplier performance issues
* Reduce stockout risks
* Optimize transportation and operational costs

---

## Tools Used

* Tableau Public
* Microsoft Excel
* Supply Chain Analytics
* Business Intelligence
* Data Visualization

---

## Dataset Information

The dataset includes:

* Order ID
* Shipment Date
* Delivery Date
* Month
* Quarter
* Year
* Region
* Warehouse Location
* Supplier Name
* Product Category
* Product Name
* Inventory Level
* Reorder Level
* Units Ordered
* Units Delivered
* Delivery Status
* Delivery Delay (Days)
* Transportation Cost
* Inventory Holding Cost
* Stockout Status
* Demand Forecast
* Actual Demand
* Inventory Turnover Ratio

---

## Dashboard KPIs

* Total Orders
* On-Time Delivery %
* Transportation Cost
* Inventory Turnover Ratio
* Stockout Count

---

## Visualizations

* Supplier Performance Analysis
* Inventory by Product Category
* Monthly Delivery Trend
* Delivery Status Distribution
* Warehouse Performance Analysis
* KPI Cards

---

## Key Insights

* Identified suppliers with delivery performance issues
* Monitored inventory levels across product categories
* Evaluated transportation costs and delivery trends
* Detected stockout risks and inventory inefficiencies
